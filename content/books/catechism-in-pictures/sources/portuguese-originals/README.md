It is a scanned image-only PDF — no text layer — and uses pre-1911 European Portuguese orthography ("Catecismo Illustrado", "doutrina christã", "introducção") that is not pt-BR.

**Recommended path:** translate from the French canonical text (`../french-originals/catechisme.txt`) using the `translate-book` skill, rather than OCR the Portuguese scan and modernize. The French is the canonical source of the work.
