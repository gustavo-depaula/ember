# Prologue — On truth

***1. What is truth?***

- 2. Whether truth is found principally in the intellect or in things?
- 3. Whether truth is only in the intellect joining and separating?
- 4. Whether there is only one truth by which all things are true?
- 5. Whether some truth besides the first truth is eternal?
- 6. Whether created truth is immutable?
- 7. Whether truth as applied to God is predicated personally or essentially?
- 8. Whether every other truth is from the First Truth?
- 9. Whether truth is in sense?
- 10. Whether there is any false thing?
- 11. Whether falsity is in sense?
- 12. Whether falsity is in the intellect?
