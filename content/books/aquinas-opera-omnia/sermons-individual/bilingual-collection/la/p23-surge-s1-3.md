# Surge.S1.3

Sed talibus dicitur Prov. XXIIII: *usquequo piger dormis?* Surge audacter quia dicitur in Iob: *rugitus leonis et vox leene* contrita est, etc. *Rugitus leonis*, id est diabolus devictus; unde in Iohannae: *confidite, ego vici mundum.* Leena etiam, id est omnis fragilitas, deponenda est quia contrita est per virtutem crucis, unde Apostolus: *vetus homo noster crucifixus est*.
